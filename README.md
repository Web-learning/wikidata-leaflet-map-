# Wikidata Leaflet Map

This project displays a Leaflet map using GeoJSON data and enriches it with data from Wikidata using Q IDs.

## Features
- GeoJSON-based markers
- Wikidata integration
- Dynamic popups

## How to run

### Option 1: Open locally
Just open `index.html` in a browser

### Option 2: Use a local server (recommended)
```bash
python -m http.server
